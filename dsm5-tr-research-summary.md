## Executive Summary

The American Psychiatric Association maintains an official repository for approved updates to the Diagnostic and Statistical Manual of Mental Disorders, Fifth Edition, Text Revision (DSM-5-TR). Updates are published as annual supplements and include changes to diagnostic criteria, text modifications, and ICD-10-CM code updates.

## Key Findings

### 1. DSM-5-TR Update Process
- **Update Frequency:** Annual supplements published each September
- **Content:** Approved changes to DSM-5-TR criteria and text
- **Format:** PDF supplements available for download
- **Coverage:** Includes diagnostic criteria changes, text revisions, and ICD-10-CM code updates

### 2. Available Updates Timeline
- **September 2025** - Most recent update
- **September 2024** - Previous year's updates  
- **September 2023** - Earlier updates
- **September 2022** - Historical updates
- **April 2022** - Special correction update

### 3. Recent ICD-10-CM Code Changes Documented

#### September 2025 Update:
- **Multiple sclerosis:** Code changed from G35 to G35.D

#### September 2023 Updates:
- **Parkinson's Disease:** 
  - Unspecified Parkinson's disease: Changed from G20 to G20.C
  - Additional codes available for dyskinesia and fluctuations (not appearing in DSM-5-TR)
- **Inadequate Housing:** Changed from Z59.1 to Z59.10

#### April 2022 Correction:
- **Opioid-Induced Anxiety Disorder:** Corrected coding discrepancy in print versions
  - Mild use disorder: Changed from F11.180 to F11.188
  - Moderate or severe use disorder: Changed from F11.280 to F11.288  
  - Without use disorder: Changed from F11.980 to F11.988

### 4. DSM-5-TR Enhancements
The DSM-5-TR incorporates:
- Fully revised text and references
- Updated diagnostic criteria
- Revised ICD-10-CM codes
- Updated coding for various disorders
